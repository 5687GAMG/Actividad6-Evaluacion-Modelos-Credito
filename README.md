
# Actividad 6 - Evaluación y Validación de Modelos

## Descripción del proyecto

Este proyecto analiza un problema financiero de clasificación binaria: predecir si un cliente incumplirá el pago de su tarjeta de crédito el siguiente mes.

## Dataset

Se utilizó el dataset Default of Credit Card Clients.

## Modelos evaluados

- Baseline
- Regresión Logística
- Random Forest

## Métricas utilizadas

- Accuracy
- Precision
- Recall
- F1-score
- AUC

## Técnicas aplicadas

- Matriz de confusión
- Ajuste de umbral
- Curva ROC y AUC
- Validación cruzada K-Fold
- Comparación con baseline
- Prueba A/B simulada
- Registro de experimentos en MLflow

## Modelo seleccionado

El modelo seleccionado fue: Random_Forest

## Justificación

El modelo fue seleccionado por presentar el mejor equilibrio entre desempeño, capacidad de detección de clientes en riesgo y utilidad para el contexto financiero.
